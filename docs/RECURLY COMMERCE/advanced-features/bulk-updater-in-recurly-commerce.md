# Overview

Bulk Updater lets you modify subscription contracts in bulk—no need to edit each subscriber individually.

# Key benefits

* **Mass updates in minutes**: Apply changes across hundreds or thousands of subscriptions with one action.
* **Consistency at scale**: Ensure uniform settings and discounts across your subscriber base.
* **Time savings**: Free your team from repetitive manual edits.

# Key details

To use Bulk Updater, navigate to the **Bulk Updater** tab of your Recurly Commerce app.
