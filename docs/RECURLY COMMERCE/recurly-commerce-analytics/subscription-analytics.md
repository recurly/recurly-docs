# Overview

Subscriptions Analytics visualizes the total number of active and renewed subscriptions in your store over any selectable date range, with side-by-side comparison to a prior period.

# Key benefits

* **Clear growth insights**: Instantly see if your subscription base is expanding or contracting.
* **Flexible date ranges**: Choose custom windows (daily, weekly, monthly) to match your reporting cadence.
* **Easy benchmarking**: The dotted-line overlay shows previous period performance for instant context.

# Key details

Select your desired time frame in the date picker, and the chart will update to show both the current period (solid line) and the comparison period (dotted line). Hover over any data point to see exact subscription counts for that date in each period.
