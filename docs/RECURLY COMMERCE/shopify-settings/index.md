# Overview

The Shopify settings section outlines how to fulfill and refund subscription orders, manage discount codes, configure shipping profiles, and leverage order tagging—all within Shopify’s interface.

# Key benefits

* **Centralized order management**: Handle fulfillment and refunds without leaving Shopify.
* **Consistent subscriber experience**: Use your existing Shopify discount and shipping tools for subscriptions.
* **Actionable insights**: Tag subscription orders for easy filtering and reporting.
