Naviga is a common backend for news and publishing customers. As long as you have Naviga Subscribe version 3.13 or more and **NCS Circ** 2018.5 plus SP1, you can utilize 1-click capabilities to create prompts within Redfast that can:

* Check whether user has an active subscription
* Create a subscription
* Start a subscription
* End a subscription

These actions are enabled through Redfast Actions integration with the API method accessing [Naviga Subscribe API](https://docs.navigaglobal.com/naviga-subscribe/additional-resources/subscribe-apis/subscribe-api). Partnership discussions are underway to enable additional actions.
