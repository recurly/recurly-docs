The [custom traits](user-traits) that are a part of your Redfast configuration can be used within a prompt. This enables you to create highly personalized prompts that appeal directly to the user.

As a simple example, if you have user’s first name and the date they became a member configured as custom traits you can use them as variables within a prompt to address the user directly like:

* “Hello Steve” or
* “Steve (member since January 2018)”

Both the title and message areas of a prompt can include dynamic user traits. You may use a dynamic variable any number of times within a prompt as appropriate to personalize the message.

Adding a dynamic variable to your prompt is as simple as clicking the **“+ Insert Variable”** link near the title area and message area when you create a new prompt. You can also add dynamic variables to an existing prompt using the same method.

## Step-by-step

* [ ] Go to **Prompts** and select your prompt.

  <Image align="center" className="border" border="true" src="https://files.readme.io/7915a13-image.png" />
