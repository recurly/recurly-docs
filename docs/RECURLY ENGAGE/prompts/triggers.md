Triggers allow you to specify the criteria of when and what you want your prompt to open on.

## Set up page and event triggers

When creating a prompt, you should identify key pages and events (buttons clicks, etc) on your site that will be used for the selected use cases. Customer success can assist you with initial setup and configuration.

**For example:**

| Item           | Detail                         |
| :------------- | :----------------------------- |
| Cancel link    | /account with id: cancel-click |
| Home screen    | /home                          |
| Account screen | /account                       |

During your platform training workshop, you will use Redfast Live to point-and-click to add pages or events triggers.

To set/select a trigger, click on the 'pencil' icon under Prompt Details, and click 'Create new trigger' to configure a new trigger or 'Select & Add trigger' to select a trigger that has already been created from the list of the triggers saved on your account.
