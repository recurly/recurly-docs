This section is where you can set up/ update your site/app configuration.

* **ID** - Your application unique identification code. You can use it to [add the Redfast tag](google-tag-manager) on your site using Google Tag Manager. 
* **API Key** - Your application API key.\
  The **ID** and **API Key** fields are pre-filled automatically. You can regenerate the API key by clicking on the arrow icon, and this will invalidate the previous key.
* **Name**(Required) - the name of your site or application. You can update this field at any time.
* **Domain**(Required) - the domain where you application is accessed. You do not need to specify a subdomain like www unless you have multiple applications under one domain. 
* **Domain Aliases** - additional domains the Redfast tag will be added to and which can be managed using this Redfast console. This can be useful when you operate multiple brands using a single backend.
* **Timezone** - the timezone of your application. This timezone will be used when scheduling your prompts (unless a user's timezone is selected) and for displaying performance data.
