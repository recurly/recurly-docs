The **Live** view displays near-realtime prompt interactions and exceptions for the prompts you currently have active. You can filter out interactions by device platform/prompt type or a user activity type, as well as search for interactions associated with a specific User ID. By switching the toggle to **Errors**, you can monitor for any action exceptions (failed  External, API or Website actions)

The Live feture can be used to:

* ensure that recently launched prompts are working properly;
* monitor live prompts during critical events;
* debug end user support issues.
