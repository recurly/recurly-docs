Launchpad is a 90-day success plan/customer learning path designed to help you build a strong foundation in Recurly, after reaching the production phase.

## It is structured into two strategic phases:

### Phase 1: Optimization

Navigate Launchpad, Phase 1: Optimization is a strategic phase of the Recurly customer success program designed to help businesses build a strong foundation in Recurly by focusing on impactful configurations. The primary purpose is to prepare and optimize Recurly setups to drive results, ensuring the system is primed before moving on to the second phase, which focuses on data expertise and mastering metrics.

### Phase 2: Mastering Metrics

Following the optimization phase, the program shifts focus to using Recurly data to uncover insights, make confident decisions, and grow smarter.
