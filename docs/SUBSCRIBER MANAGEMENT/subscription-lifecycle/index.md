# Definition

<div class="rp-definition">The subscription lifecycle encompasses every stage and action in a subscriber's tenure — from initial setup through modifications, pauses, postponements, and eventual cancellation or expiry. Recurly provides dedicated tools for each stage, with API and Admin UI support throughout.</div>
