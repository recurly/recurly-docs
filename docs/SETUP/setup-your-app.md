Start by telling us the name of your application or website, ex. `Demo`. Add the domain where you application is accessed, ex. `demo.redfast.com`. You do not need to specify a subdomain like `www` unless you have multiple applications under one domain.

Domain aliases are additional domains that can be managed using a single Redfast console. This can be useful when you operate multiple brands using a single backend.
