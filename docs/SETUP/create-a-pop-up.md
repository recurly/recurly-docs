In this guide we will be creating a pop up. The instructions for creating interstitial, notification, video prompt are exactly the same.

### Go to Prompts > New Prompt Button (Plus sign)

<Image align="center" className="border" border={true} src="https://files.readme.io/c46f2ae-Screenshot_2024-05-23_at_16.31.29.png" />

### Select "Desktop and Mobile" since we are creating a prompt for the browser. Then select the pop up.

<Image align="center" className="border" border={true} src="https://files.readme.io/66c5557-Screenshot_2024-05-23_at_16.32.40.png" />

### Add a name and description

<Image align="center" className="border" border={true} src="https://files.readme.io/6d1cd5a-Screenshot_2024-05-23_at_16.34.02.png" />

### Add one or more segments

Adding the "Test Users" segment allows you to view the prompt as if you were a "AVOD user." If you haven't created any segments yet you can just add the "Test Users" segment.

<Image align="center" className="border" border={true} src="https://files.readme.io/c389c63-Screenshot_2024-05-23_at_16.35.19.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/9acb5c6-Screenshot_2024-05-23_at_16.36.08.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/8b788cc-Screenshot_2024-05-23_at_16.37.00.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/74f4509-Screenshot_2024-05-23_at_16.37.51.png" />

### Add a trigger

Adding a trigger allows you to decide where you want to show your prompt. In this case we will select "All Pages." Any user who comes to the site will immediately see the popup, regardless of what page they landed on.

<Image align="center" className="border" border={true} src="https://files.readme.io/035de64-Screenshot_2024-05-23_at_16.38.49.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/7bfe1b7-Screenshot_2024-05-23_at_16.39.44.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/371bb5f-Screenshot_2024-05-23_at_16.40.39.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/21823b1-Screenshot_2024-05-23_at_16.41.38.png" />

### Set a limit (optional)

If you have a budget or want to set a limit on how many prompts can be claimed you can do it here. There are multiple types of limits you can choose from.

<Image align="center" className="border" border={true} src="https://files.readme.io/f44ce1a-Screenshot_2024-05-23_at_16.43.11.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/ecd3c1c-Screenshot_2024-05-23_at_16.44.28.png" />

### Set a schedule (optional)

If you want to schedule your prompt to run at a certain time you can do so by clicking Schedule

<Image align="center" className="border" border={true} src="https://files.readme.io/3ea543f-Screenshot_2024-05-24_at_16.13.26.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/8779e53-Screenshot_2024-05-24_at_16.14.16.png" />

### Set an action (optional)

If you want to redirect a user to a new page after they accept this prompt you can do so by clicking Website Actions

<Image align="center" className="border" border={true} src="https://files.readme.io/2d57a20-Screenshot_2024-05-24_at_16.16.39.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/e353f8a-Screenshot_2024-05-24_at_17.13.47.png" />

### Select edit to design the prompt

<Image align="center" className="border" border={true} src="https://files.readme.io/e5df649-Screenshot_2024-05-24_at_17.15.01.png" />

### Design the prompt

This is where the magic happens. Feel free to tinker with the settings and see your changes live in the preview at the top. If you don't want to change any settings you can leave it as is and click "Continue."

Background image size varies by device. Click here for a [zip file with sample background images](https://docs.redfast.com/images/samples.zip).

<Image align="center" className="border" border={true} src="https://files.readme.io/67fe571-Screenshot_2024-05-24_at_17.20.04.png" />

### Preview your prompt

Once you have saved your changes. You can preview your prompt on your live site.

<Image align="center" src="https://files.readme.io/23d54d9-Screenshot_2024-05-24_at_17.23.01.png" />

<Image align="center" className="border" border={true} src="https://files.readme.io/d5b09d4-Screenshot_2024-05-24_at_17.23.34.png" />

### That's it!

Once you are satisfied with your design, hit 'Save' to apply the changes.  Your prompt has been created and you can set its status to 'Start'. Once it's live, if you open your webapp you will now see the prompt.

<Image align="center" className="border" border={true} src="https://files.readme.io/72e4d57-Screenshot_2024-05-24_at_17.24.40.png" />

### Test the prompt

A prompt can be tested before it is published to all users. These tests are useful for refining triggers, creative, schedules, and actions. You can learn to do so by following the [test user guide](test-users).
