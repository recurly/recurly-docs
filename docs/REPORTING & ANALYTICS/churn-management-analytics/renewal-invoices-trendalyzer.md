# Overview

### Required plan

This feature is available to all customers on any Recurly subscription plan.

### Prerequisites

- Users must have Analytics user role permission.

# Definition

The renewal invoices trendalyzer dashboard provides insights into all renewal invoices, helping you assess the effectiveness of your payment methods and gateway strategy to boost success rates. With this tool, you can track trends at the daily, weekly, or monthly level across any date range. Users can arrange various dimensions—such as Payment Type, Brand, Gateway, and Invoice Outcome—in their preferred order and select from key metrics, including Total Paid Count, Total Count, Total Paid Amount, Renewal Rate by Count (%), and Renewal Rate by Amount (%).

# Key benefits

- **Informed decision-making**: Use real-world data to set realistic goals and optimize subscription growth and customer retention strategies.
- **Forecasting and planning**: Improve forecasting and long-term planning for setting achievable growth targets and allocating resources effectively.
- **Customizable data views**: Dynamically pivot data for optimal presentation, select specific metrics to trend, and analyze various date ranges at the daily, weekly, or monthly level.

# Key details

This breakout dashboard allows you to adjust the pivot table to view different fields in various orders and switch between metrics like Total Paid Count, Total Count, Total Paid Amount, Renewal Rate by Count (%), and Renewal Rate by Amount (%). The top chart displays the primary dimension over time, and the table provides a breakdown of the primary dimension by the selected breakout dimensions.

- **Timeframe**: Analyze data across different monthly timeframes.
- **Invoice Creation Date**: Select a specific date range, up to the last 18 months, for dashboard data. Dropdown menus allow you to choose the start and end dates.
- **Primary Dimension**: Focus on a specific metric by selecting between Payment Type, Brand, Gateway, or Invoice Outcome.
- **Dimension Breakout 1**: Choose an additional dimension from the list, or view all dimensions.
- **Dimension Breakout 2**: Choose another dimension from the list, or view all dimensions.
- **Metric**: Select the key metric to analyze, choosing from Total Paid Count, Total Count, Total Paid Amount, Renewal Rate by Count (%), or Renewal Rate by Amount (%).
